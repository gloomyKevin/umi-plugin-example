[![NPM downloads](http://img.shields.io/npm/dm/myp.svg?style=flat)](https://npmjs.org/package/myp)



## Install

```bash
# or yarn
$ npm install
```

```bash
$ npm run build --watch
$ npm run start
```

## Usage

Configure in `.umirc.js`,

```js
export default {
  plugins: [
    ['umi-plugin-my'],
  ],
}
```
